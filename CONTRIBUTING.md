# 🤝 Wie man einen Kommentar hinzufügt

Danke, dass du Blue Blocker magst! 💙 Hier ist die einfache Anleitung, um dein Feedback zu geben:

---

## 📋 Schritt-für-Schritt Anleitung

### **Schritt 1: Repository forken**
1. Gehe zum [Blue Blocker Repository](https://github.com/jamming-peter96/Blue_blocker)
2. Klicke auf **"Fork"** (oben rechts)
3. Jetzt hast du eine Kopie in deinem Account

---

### **Schritt 2: `comment_here.md` bearbeiten**
1. Öffne in **deinem Fork** die Datei `comment_here.md`
2. Klicke auf **"Edit"** (Stift-Symbol)
3. Füge deinen Kommentar hinzu:

```
### [TAG_NAME]
- **@dein_username** (TT.MM.YYYY): "Dein Kommentar hier..."
```

#### **Beispiel:**
```
### [features]
- **@dein_name** (05.07.2026): "Großartiges Projekt! Würde noch eine Dark Mode Option schön finden."
```

#### **Verfügbare Tags:**
- `performance` - Geschwindigkeit und Performance
- `design` - Design und Benutzeroberfläche  
- `features` - Neue Funktionen oder Feature-Anfragen
- `documentation` - Dokumentation
- `security` - Sicherheitsaspekte
- `speed` - Optimierungen
- `bug` - Gefundene Fehler
- `improvement` - Allgemeine Verbesserungsvorschläge

---

### **Schritt 3: Commit die Änderung**
1. Scroll nach unten
2. Gib eine Commit-Nachricht ein: `Add comment: [dein Kommentar Kurzbeschreibung]`
3. Klicke **"Commit changes"**

---

### **Schritt 4: Pull Request erstellen**
1. Du siehst jetzt einen grünen Button: **"Compare & pull request"**
2. Klicke drauf
3. Schreib eine kurze Beschreibung (optional)
4. Klicke **"Create pull request"**

---

### **Schritt 5: Fertig! ✨**
- Der Repository-Owner (jamming-peter96) wird deine Änderung reviewen
- Nach dem Merge wird dein Kommentar **automatisch auf der README** angezeigt!
- GitHub Actions aktualisiert alles automatisch 🤖

---

## 💡 Tipps

- **Dein GitHub-Username:** Nutze `@dein_github_username`
- **Datum:** Format `TT.MM.YYYY` (z.B. `05.07.2026`)
- **Kommentar:** Sei konstruktiv und freundlich! 😊
- **Ein Kommentar pro PR:** Am besten einen Kommentar pro Pull Request

---

## ❓ Fragen?

Wenn etwas nicht klappt, öffne ein [Issue](https://github.com/jamming-peter96/Blue_blocker/issues) oder schreib einen Kommentar!

Danke für dein Feedback! 🙌
